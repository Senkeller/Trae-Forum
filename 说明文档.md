# TRAE Forum Flutter App - UI 组件开发文档

## 项目规划

### 目标
开发一套完整的 Flutter UI 组件库，支持 Feed 流、评论、用户等核心功能模块。

### 组件架构
```
lib/presentation/widgets/
├── common/          # 通用组件
├── feed/            # Feed 相关组件
├── comment/         # 评论相关组件
└── user/            # 用户相关组件
```

## 实施方案

### 技术栈
- **UI 框架**: Flutter Material Design 3
- **主题**: 支持深色/浅色模式切换
- **图片加载**: cached_network_image
- **骨架屏**: shimmer

### 组件清单

#### 1. 通用组件 (common/)
| 组件名 | 功能 | 依赖 |
|--------|------|------|
| loading_widget.dart | 加载指示器、骨架屏 | shimmer |
| empty_widget.dart | 空状态提示 | - |
| error_widget.dart | 错误状态提示（可重试） | - |
| cached_image.dart | 带缓存的图片组件 | cached_network_image |

**LoadingWidget** - 加载状态组件
- 支持圆形进度指示器
- 支持自定义提示文字和颜色

**SkeletonWidget** - 骨架屏组件
- 支持自定义宽高和圆角
- 自动适配深色/浅色主题

**ListSkeletonWidget** - 列表骨架屏
- 模拟列表项加载效果
- 可配置列表项数量

**CardSkeletonWidget** - 卡片骨架屏
- 模拟 Feed 卡片加载效果
- 支持图片区域占位

**EmptyWidget** - 空状态组件
- 支持自定义图标、标题、描述
- 支持操作按钮
- 预置多种场景：搜索为空、网络为空、通知为空等

**ErrorWidget** - 错误状态组件
- 支持自定义图标、标题、描述
- 支持重试按钮
- 预置多种场景：网络错误、服务器错误、404等

**CachedImage** - 带缓存的图片组件
- 使用 cached_network_image 实现
- 支持圆角、圆形裁剪
- 支持占位图和错误图
- 集成 shimmer 骨架屏效果

#### 2. Feed 组件 (feed/)
| 组件名 | 功能 | 依赖 |
|--------|------|------|
| feed_card.dart | 动态卡片容器 | 组合其他组件 |
| feed_content.dart | 动态内容（文字、九宫格图片） | cached_image |
| feed_actions.dart | 动态操作栏（点赞、评论、分享） | - |
| feed_author.dart | 作者信息（头像、昵称、时间） | user_avatar |

**FeedCard** - 动态卡片
- 组合作者、内容、操作栏
- 支持点赞、评论、分享、收藏回调
- 支持关注按钮

**FeedCardData** - 动态数据模型
- 包含完整的动态信息字段

**FeedAuthor** - 动态作者信息
- 展示头像、昵称、等级
- 显示发布时间和地点
- 支持关注按钮

**FeedContent** - 动态内容
- 支持文字内容（展开/收起）
- 支持九宫格图片布局
- 支持单图、四图、多图自适应布局

**FeedActions** - 动态操作栏
- 点赞、评论、分享、收藏按钮
- 显示数量统计
- 支持状态切换动画

#### 3. 评论组件 (comment/)
| 组件名 | 功能 | 依赖 |
|--------|------|------|
| comment_card.dart | 评论卡片 | user_avatar |
| comment_input.dart | 评论输入框 | - |

**CommentCard** - 评论卡片
- 展示评论内容和作者信息
- 支持子评论（回复）展示
- 支持点赞、回复操作
- 支持评论图片展示

**CommentData** - 评论数据模型
- 包含评论完整信息
- 支持嵌套回复

**CommentInput** - 评论输入框
- 支持多行输入
- 支持表情、图片、@用户按钮
- 字数统计
- 回复提示

**BottomCommentInput** - 底部评论栏
- 固定在底部
- 集成点赞、评论、分享、收藏操作

#### 4. 用户组件 (user/)
| 组件名 | 功能 | 依赖 |
|--------|------|------|
| user_avatar.dart | 用户头像（圆形，支持点击） | cached_image |
| user_name.dart | 用户名显示（带等级标识） | - |
| follow_button.dart | 关注按钮（关注/已关注状态） | - |

**UserAvatar** - 用户头像
- 支持多种尺寸：small(40)、medium(48)、large(56)、xlarge(80)
- 支持在线状态指示器
- 支持认证标识（V标）
- 支持点击跳转

**UserAvatarGroup** - 用户头像组
- 重叠展示多个头像
- 支持数量限制和溢出提示

**UserName** - 用户名显示
- 支持用户等级标识（VIP、SVIP、管理员等）
- 支持认证标识
- 支持点击跳转

**UserLevel** - 用户等级枚举
- normal、vip、svip、admin、moderator、official

**FollowButton** - 关注按钮
- 支持多种状态：未关注、已关注、互相关注、加载中
- 支持小尺寸版本
- 支持确认对话框版本

**FollowCountWidget** - 关注数量显示
- 展示关注数和粉丝数
- 支持数字格式化（k、w）

## 设计规范

### 颜色规范
- 使用 AppTheme 中定义的颜色
- 支持深色/浅色主题自动切换
- 功能色：success、warning、error、info
- 点赞色：Color(0xFFE91E63)
- 收藏色：Color(0xFFFF9800)

### 尺寸规范
- 圆角：8px（按钮）、12px（卡片）、16px（对话框）、20px（底部弹窗）
- 间距：8px（小）、12px（中）、16px（大）、24px（超大）
- 头像：40px（小）、48px（中）、56px（大）、80px（超大）

### 字体规范
- 标题：16-18px，FontWeight.w600
- 正文：14px，FontWeight.normal
- 辅助：12px，FontWeight.w500

## 代码规范
- 所有函数必须包含函数级注释（功能描述、参数说明、返回值）
- 使用 const 构造函数优化性能
- 组件参数使用 required 标记必填项
- 支持 onTap、onRetry 等回调函数
- 使用 Material Design 3 风格
- 支持深色/浅色主题

## 使用示例

### FeedCard 使用
```dart
FeedCard(
  data: FeedCardData(
    id: '1',
    username: '用户名',
    avatarUrl: 'https://example.com/avatar.jpg',
    text: '动态内容',
    images: ['https://example.com/image.jpg'],
    likeCount: 100,
    commentCount: 20,
  ),
  onLike: () {},
  onComment: () {},
)
```

### CommentCard 使用
```dart
CommentCard(
  data: CommentData(
    id: '1',
    username: '评论者',
    content: '评论内容',
    publishTime: DateTime.now(),
  ),
  onLike: () {},
  onReply: () {},
)
```

### UserAvatar 使用
```dart
UserAvatar(
  avatarUrl: 'https://example.com/avatar.jpg',
  size: 48,
  showOnlineStatus: true,
  isOnline: true,
)
```
